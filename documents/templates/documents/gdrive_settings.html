Setup Instructions

  1. Go to the Google Cloud Console and create or select a project.
  2. Enable the Google Drive API and Google Picker API.
  3. Go to APIs & Services → Credentials.
  4. Create an OAuth 2.0 Client ID (Web application type).
  5. Add your callback URL as an authorized redirect URI:
    {{ request.scheme }}://{{ request.get_host }}{% url 'documents:gdrive_callback' %}
  6. Copy the Client ID and Client Secret into the fields above.
  7. Create an API Key and restrict it to the Picker API.
  8. Go to IAM & Admin → Settings and copy the Project number (numeric).
